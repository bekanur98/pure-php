# Как запустить мигарцию?
mysql: 8.0
php: 7.3

Если вы уже настроили config/database.php для вашего mysql и уже настроен php7.3 также настроена nginx или apache, то переходим по след пути 
```bash 
http://localhost:80/migrations/index.php
```
